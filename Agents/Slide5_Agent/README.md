# Slide5_Agent: Candidate Companies

## Purpose
Identify candidate companies with top 3 for Core/Layer/Edge tiers, descriptors, justifications, and sources.

## Output Format
1. **Core Company**: Category leader in foundational infrastructure
2. **Layer Company**: Best-in-class middleware/tooling/AI layer
3. **Edge Company**: Distribution anchor/vertical platform/workflow owner

Each includes company name, descriptor, justification, and tier mapping.

## Usage
```bash
python agent.py "<sector>" "<region>"
```

## Output
Results saved to `../../Outputs/slide5_{sector}_{timestamp}.md`
