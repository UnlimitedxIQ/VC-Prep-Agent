# Slide3_Agent: Macro-Market Thesis

## Purpose
Create macro-market thesis with focus sub-segment, sharp opinion with contrast, and thought-leadership hook.

## Output Format
1. **Focus**: Specific sub-segment being targeted
2. **The Opinion**: Sharp take with contrast showing differentiated view
3. **Thought Leadership Hook**: Memorable 2-5 word phrase + defining sentence

## Usage
```bash
python agent.py "<sector>" "<region>"
```

## Output
Results saved to `../../Outputs/slide3_{sector}_{timestamp}.md`
